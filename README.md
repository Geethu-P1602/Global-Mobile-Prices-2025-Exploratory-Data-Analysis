# Global-Mobile-Prices-2025-Exploratory-Data-Analysis
This project analyzes global smartphone price data to understand pricing trends and how different features like brand, RAM, storage, battery, and camera influence mobile phone prices in 2025.

## Tools and Technologies  
Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn.

## File Description  
- `Global_Mobile_Prices_2025.ipynb`  
  Contains data loading, cleaning, visualization, and analysis of mobile phone prices.

## Steps Followed  

1. Data Loading  
   The dataset is loaded and basic information is examined.

2. Data Cleaning  
   Missing values and inconsistent data are handled.

3. Exploratory Data Analysis  
   Price distribution, brand-wise comparison, and feature-wise analysis are performed using graphs and charts.

4. Visualization  
   Different plots are used to understand trends and relationships in the data.

## How to Run the Project  

1. Download or clone this repository.
2. Install Python and Jupyter Notebook.
3. Open the file `Global_Mobile_Prices_2025.ipynb` in Jupyter Notebook.
4. Run all cells to view the analysis.

## Conclusion  
This EDA project helps in understanding global smartphone pricing patterns
